Description

ReactiveXD is an object oriented reactive programming language with explicit support for distribution and explicit management of time. We argue that reactive frameworks with these features facilitate the design, implementation and evolution of modern massively distributed and dynamic applications.

Project structure

The language consists on 5 main projects described as follows:

DirectoryDescription
co.edu.escuelaing.reactivexdGrammar definition and all language-specific components (parser, lexer, linker, validation, etc.)
co.edu.escuelaing.reactivexd.testsUnit tests for the language
co.edu.escuelaing.reactivexd.idePlatform-independent IDE functionality
co.edu.escuelaing.reactivexd.uiThe Eclipse editor and other functionality
co.edu.escuelaing.reactivexd.ui.testsUnit tests for the Eclipse editor

The framework uses rewriting logic and includes a model of the distributed base application, a model of observables and observers, and a model for complex predicate constructs supporting explicit time manipulation.

Requirements

    java -version

  2. Maven

    mvn -v

    3. Installation

      cd co.edu.escuelaing.reactivexd.parent-master
